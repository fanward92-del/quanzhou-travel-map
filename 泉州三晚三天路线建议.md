# 泉州三晚三天路线建议

按当前日期 2026-06-15 推算，你说的周三到周六是：

- 周三：2026-06-17 晚到泉州
- 周四：2026-06-18 古城
- 周五：2026-06-19 石狮海岸线
- 周六：2026-06-20 下午返程

如果不是本周出行，路线逻辑仍可用，但开放时间、天气、潮汐和交通要重新看。

## 结论

原计划大方向合理，但周五石狮太满。黄金海岸、洛伽寺、红塔湾、永宁古镇、观音山全做完，会变成“下车拍照式打卡”，遇到雨天/堵车就会崩。

建议保留：

1. 必须保留：黄金海岸线、天后宫、关岳庙。
2. 周四古城：开元寺、西街、钟楼、清净寺、关岳庙、天后宫都能玩完。
3. 周五石狮：洛伽寺 + 黄金海岸为核心，红塔湾看天气，永宁古镇做午后避晒/避雨点，观音山机动。
4. 周六：洛阳桥上午去，下午返程最稳。

## 推荐路线

### 周三晚：入住 + 西街夜逛

时间不宜排满：

- 入住古城附近，建议西街/钟楼/开元寺附近。
- 晚上逛西街、钟楼夜景。
- 只吃顺路小吃，不排长队。

### 周四：古城主线

推荐顺序：

1. 08:30-10:10 开元寺
2. 10:10-10:40 西街走到钟楼
3. 11:00-11:35 清净寺
4. 11:40-12:20 通淮关岳庙
5. 12:20-13:50 午餐/休息
6. 14:10-15:20 天后宫 + 德济门遗址
7. 16:00 后回西街/中山路自由逛

这样走的好处是：早上先看开元寺，清净寺和关岳庙几乎相邻，下午再去天后宫，不来回折返。

### 周五：石狮海岸线

推荐顺序：

1. 08:30 古城出发
2. 10:00-11:10 洛伽寺
3. 11:10-12:00 黄金海岸
4. 12:30-14:30 永宁古镇/午餐
5. 15:00-16:10 红塔湾
6. 观音山：只作为机动，天气好、有车、有体力再补
7. 17:00 左右返程

建议周五打车或包车。公共交通能到，但会吃掉太多时间。

如果下午下雨或雷阵雨，直接删红塔湾/观音山，保留洛伽寺、黄金海岸、永宁古镇。

### 周六：洛阳桥 + 返程

推荐顺序：

1. 08:30 退房/寄存行李
2. 09:20-10:30 洛阳桥
3. 11:20-12:40 回古城午餐/取行李
4. 下午返程

如果回程早于 14:00，周六不要再加新景点；如果回程在 16:00 以后，可以补古城小巷或吃一顿午饭。

## 天气提醒

按 2026-06-17 至 2026-06-20 的天气预报，泉州这几天以阴雨和短时阵雨为主；2026-06-19 石狮下午有雷阵雨风险。海边行程最好前置到上午，下午不要在海岸线硬等。
