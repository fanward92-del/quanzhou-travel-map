# 泉州路线 + 美食安排

原则：古城吃古城，石狮吃石狮；必须吃的店都放进路线，但不把石狮小吃塞进古城步行日。

住宿点：汉庭酒店（泉州西街中山路店），鲤城区胭脂巷59号。携程页面提取到的高德坐标为 118.588489, 24.903863。下面距离均为直线距离，实际步行/打车以当天导航为准。

|---|---:|---|
| 水门国仔面线糊 | 0.3 公里 | 周四早餐 |
| 老字号肉燕汤 | 0.6 公里 | 周四傍晚点心 |
| 秉正堂四果汤 | 0.4 公里 | 周四天后宫后 |
| 美美阿姨醋肉 | 1.3 公里 | 周四傍晚点心 |

## 周三晚

| 类型 | 安排 | 说明 |
|---|---|---|
| 晚饭 | 斯丹姜母鸭 | 如果到得不晚就吃；到得太晚则挪到周四晚饭。 |
| 点心 | 丛甜牛奶冰 | 饭后或西街夜逛时吃。 |
| 游玩 | 西街 + 钟楼夜景 | 不再加寺庙，早点休息。 |

## 周四古城

| 类型 | 安排 | 说明 |
|---|---|---|
| 早餐 | 水门国仔面线糊 | 当天选离酒店近且营业的分店；官方资料含美食街155号、水门巷21号，高德另见中山中路118/120号。 |
| 上午 | 开元寺 → 西街 → 钟楼 → 清净寺 → 关岳庙 | 这几个点已拆成独立地图点并按顺序连线；关岳庙是必去点，清净寺和关岳庙相邻。 |
| 午饭 | 阿秋牛排总店 | 阿秋/阿波二选一，优先阿秋；排队太久改阿波牛肉。 |
| 下午 | 天后宫 → 德济门遗址 | 两个点已拆开连线；天后宫是必去点，德济门就在旁边顺路看。 |
| 下午点心 | 秉正堂四果汤 | 天后宫后顺路吃。 |
| 傍晚点心 | 老字号肉燕汤、美美阿姨醋肉 | 都按小份/分食，不要当两顿正餐。 |
| 补漏 | 铁记醋肉 | 目前未核到可靠门牌；当天能定位到就插入古城傍晚，找不到则至少保留美美阿姨醋肉。 |
| 晚饭 | 斯丹姜母鸭或古城自由晚饭 | 周三没吃斯丹，周四晚必须补。 |

## 周五石狮

| 类型 | 安排 | 说明 |
|---|---|---|
| 早餐 | 王川花生汤 | 到石狮后先吃。 |
| 点心 | 马脚桥鱼肉羹、约记肉粽、福记蚵仔煎 | 这三家按小吃拼盘处理，建议小份分着尝。 |
| 游玩 | 洛伽寺 + 黄金海岸 | 黄金海岸线是重点，先完成。 |
| 午后 | 永宁古镇 | 避晒/避雨，顺便慢走。 |
| 机动 | 红塔湾/观音山 | 天气好、有车、有体力再补。 |
| 晚饭 | 绿岛海鲜酒楼/绿岛酒店 | 吃海鲜优先搜绿岛海鲜酒楼，核到地址为石狮九二路447-451号。 |

## 周六返程

| 类型 | 安排 | 说明 |
|---|---|---|
| 早餐 | 古城补漏早餐 | 没吃到水门国仔就补；吃过就简单吃。 |
| 上午 | 洛阳桥 | 上午去，给下午返程留余量。 |
| 午饭 | 古城轻午餐/补漏点心 | 只补一个漏掉的点，不要硬吃全套。 |

## 必吃完成表

- 斯丹姜母鸭：周三晚或周四晚。
- 水门国仔面线糊：周四早餐。
- 阿秋或阿波牛肉：周四午饭，优先阿秋。
- 秉正堂四果汤：周四天后宫后。
- 丛甜牛奶冰：周三晚或周四补。
- 老字号肉燕汤：周四傍晚。
- 美美阿姨醋肉：周四傍晚。
- 铁记醋肉：周四古城补漏，需当天定位。
- 马脚桥鱼肉羹、福记蚵仔煎、约记肉粽、王川花生汤、绿岛海鲜酒楼/绿岛酒店：全部放周五石狮。
