# 泉州路线 + 美食安排

原则：古城吃古城，石狮吃石狮；必须吃的店都放进路线，但不把石狮小吃塞进古城步行日。周三晚自驾到达后只吃一顿主菜和轻夜宵，周四集中走古城，周五再单独开车去石狮。

住宿点：汉庭泉州西街新门街酒店，鲤城区新门街787号2栋。携程/Trip 等平台也常显示为“汉庭酒店（泉州西街开元寺店）”。它和“汉庭酒店（泉州西街中山路店，胭脂巷59号）”不是同一家。携程页面提取到的高德坐标为 118.575828, 24.911484。下面距离均为直线距离，实际步行/打车以当天导航为准。

## 周三晚：到达后只吃姜母鸭，夜景轻轻走

| 类型 | 安排 | 说明 |
|---|---|---|
| 入住 | 汉庭泉州西街新门街酒店 | 到达后先办入住，行李放好再出门。 |
| 晚饭 | 斯丹姜母鸭 | 这顿确定安排在周三晚，吃完不要再硬排大景点。 |
| 游玩 | 西街 + 钟楼夜景 | 只做夜景和轻逛，累了就直接回酒店。 |
| 点心 | 丛甜牛奶冰 | 饭后甜点，适合从西街慢慢回酒店前后吃。 |
| 夜宵 | 铁霞烧烤 | 只点小份花生酱烧烤尝味；太累就取消。具体门牌当天用高德再确认。 |

## 周四古城：先城南，再阿秋，下午回东街西街

| 类型 | 安排 | 说明 |
|---|---|---|
| 早餐 | 水门国仔面线糊 | 早餐后直接从城南片区开始，不先折回开元寺。 |
| 上午 | 天后宫 → 德济门遗址 | 天后宫是必去点，德济门就在旁边，先完成这两个。 |
| 上午点心 | 秉正堂四果汤 | 放在天后宫后最顺，当作休息点，不要点太多。 |
| 上午 | 清净寺 → 通淮关岳庙 | 两个点距离近，关岳庙是必去点。 |
| 午饭 | 阿秋牛排总店 | 优先阿秋；若排队太久，改阿波牛肉。 |
| 下午 | 东街 → 钟楼 → 西街 → 开元寺 | 午饭后打车回古城，从东街接回西街，最后到开元寺和酒店附近。 |
| 傍晚点心 | 老字号肉燕汤、美美阿姨醋肉 | 都按小份/分食处理，不要当两顿正餐。 |
| 补漏 | 铁记醋肉 | 目前未核到可靠门牌；当天能定位到就插入古城傍晚，找不到则至少保留美美阿姨醋肉。 |
| 晚饭 | 侯阿婆肉粽（东街总店） | 周三已吃姜母鸭，周四晚饭选更轻、更有泉州特色的肉粽；太饱可外带。 |

## 周五石狮：自驾单独排，别把海岸线跑散

| 类型 | 安排 | 说明 |
|---|---|---|
| 早餐 | 酒店附近简单吃 | 不为早餐绕路，直接准备开车去石狮。 |
| 早午饭 | 马脚桥鱼肉羹 | 到石狮第一站，按早午饭吃，建议小份。 |
| 小吃拼盘 | 约记肉粽、福记蚵仔煎、王川花生汤 | 三个都只做拼盘尝味，吃不下就外带或跳过一个。 |
| 游玩 | 洛伽寺 + 黄金海岸 | 这是周五核心，优先完成，不再来回折返。 |
| 机动 | 红塔湾/永宁古镇/观音山三选一 | 天气好选红塔湾，想慢走选永宁古镇；观音山只在体力、天气和停车都合适时补。 |
| 晚饭 | 绿岛海鲜酒楼/绿岛酒店 | 吃海鲜优先搜绿岛海鲜酒楼，核到地址为石狮九二路447-451号。 |
| 返程 | 开车回泉州古城酒店 | 不换酒店，晚上回汉庭新门街。 |

## 周六返程：洛阳桥 + 一个补漏

| 类型 | 安排 | 说明 |
|---|---|---|
| 早餐 | 古城补漏早餐 | 没吃到水门国仔就补；吃过就酒店附近简单吃。 |
| 上午 | 洛阳桥 | 上午去最稳，给下午返程留余量。 |
| 午饭 | 古城轻午餐/补漏点心 | 只补一个漏掉的点，不要硬吃全套。 |
| 下午 | 取行李返程 | 按实际车站/自驾路线调整时间。 |

## 必吃完成表

- 斯丹姜母鸭：周三晚饭。
- 水门国仔面线糊：周四早餐。
- 阿秋或阿波牛肉：周四午饭，优先阿秋。
- 秉正堂四果汤：周四天后宫后。
- 丛甜牛奶冰：周三晚甜点。
- 老字号肉燕汤：周四傍晚。
- 美美阿姨醋肉：周四傍晚。
- 铁记醋肉：周四古城补漏，需当天定位。
- 马脚桥鱼肉羹、福记蚵仔煎、约记肉粽、王川花生汤、绿岛海鲜酒楼/绿岛酒店：全部放周五石狮。
